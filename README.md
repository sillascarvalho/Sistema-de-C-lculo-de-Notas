# Sistema-de-C-lculo-de-Notas
O sistema de cálculo de notas para alunos é uma aplicação web simples e funcional desenvolvida para facilitar o gerenciamento de avaliações escolares. Ele permite que o usuário insira o nome do aluno e uma lista de notas, separadas por vírgulas. Com base nesses dados, o sistema calcula automaticamente a média das notas e exibe o resultado.
